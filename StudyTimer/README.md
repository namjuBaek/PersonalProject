# StudyTimer / 스터디 타이머
> 학생들이 자주 사용하는 스터디 타이머를 휴대폰으로 사용할 수 있도록 앱으로 구현

<br>

##### 주요기능

1. Clock
2. Count-up 
3. Count-down
4. Background 실행
5. 가로버전 지원

<br>

<br>

#### Clock

---

1. 위치 기준 현재 시각 조회 기능 (시계)

<br>

<br>

#### Count_up

---

1. 공부 시간 측정

<br><br>

#### Count_down

---

1. 카운트 할 시간 설정
2. 시작 / 일시정지 / 정지 기능이 구현되어야 함

##### CountDownTimer 함수 사용

- 재생할 시간과 텀을 인자로 받아 카운트 다운 실행

- pause 기능은 제공하지 않아 **직접 구현**해야 함
